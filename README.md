## Getting Started

### Prerequisites

For this project you need:

```
Nodejs
```

### Installing

Go to /insurance-quote/insurance-quote and run:

```
npm install
```

## Running React

In the same folder run:

```
npm start
```

Now you can use Insurance Calculator!
